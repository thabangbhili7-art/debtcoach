# DebtCoach

DebtCoach is a WhatsApp-based assistant that helps users organise debts, track payments, and estimate how long it will take to become debt-free.

## What does DebtCoach do?
- Helps users list their debts
- Tracks payments they make
- Shows remaining balances
- Estimates a payoff timeline based on a monthly budget

## How users interact
Users start a conversation by sending a message such as:
- “Hi”
- “Hello”

The assistant then asks for consent before collecting any debt information.

## Example messages users may receive
- Debt summaries
- Payment confirmations
- Remaining balance updates

Example commands:
- **DEBTS** – view current debts
- **PAID 500** – log a payment
- **SUMMARY** – view total balance and payoff estimate
- **RESET** – clear all data

## Consent & Opt-out
Users must explicitly consent before using the service.

Users can stop using the service at any time by:
- Sending **RESET**
- Or stopping messages on WhatsApp

## Data handling
DebtCoach stores only the minimum information required to provide the service:
- Phone number
- Debt balances
- Payment history

No data is shared with third parties.
